Smart Irrigator

  • Aim of this project was to develop an Automatic Irrigation System with the help of arduino-uno.
  • Main purpose of this project is to detect moisture level of the soil and turn the water pump on and off accordingly.
  • esp8266 was used to send soil related data to our application.
  • 3rd party application named RemoteXY was used to implement IOT features. Like showing various statistics and providing manual control over the motor along with the Automatic control.
